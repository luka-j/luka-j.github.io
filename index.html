Latest Projects

Stories

A mobile app and a programming language for playing, creating and editing interactive stories

App

Designed having programming greenhorns in mind, Stories attempts to bring programming into realm of games. By creating a domain-specific language for creating interacive stories and designing a runtime on Android, everything is made mobile. Apart from runtime, the app also contains a visual editor, a code editor and a marketplace where you can download others' and publish your own stories.

The app was ranked top five at seventh mt:s App konkurs and won special prize for innovativity at Telekom Srbija's Regional App Challenge (Serbian: Regionalni app izazov).

The language and a platform-agnostic interpreter written in Java are open-source and can be found on my GitHub

Upis suiteOpen Source

Three programs designed to gather data about Serbian primary schools which are freely available online. It collects results of the final exam and final grades for every student which graduated. Scraper is an extensible program used for scraping the data, basic data manipulation and simulating enrollment process, optionally with different weighting criteria. Web server pulls data from scraper, builds a database and presents an interface for ploting the data, using a special query language built on top of SQL. Desktop client uses the server as an API and provides point-and-clict interface for creating queries. It was a subject of my high school graduation thesis (in Serbian), which was recognized as one of that year's best theses in the field of informatics.
